Задача:

Сверстать страничку по заданному дизайну.
Дизайн (psd ~4.4 Mb) - это дизайн блока в "максимальном" виде
Скриншот результата (png ~515 kb) - это должно получиться!
Резина (png ~900 kb) - так должно тянуться при изменении размеров браузера

Критерии:

  • "Резиновая" верстка без min-width (возможно min-width: 400px для одной обертки)
  • Правильное "поведение" блоков согласно описанию
  • Точная реализация дизайна
  • Graceful degradation для IE lte 9
  • Кроссбраузерность (IE 8+, FF 3.6+, Opera 11+, Chrome 16+)
  • Использование методологии БЭМ будет плюсом

Результат:

Архив содерщащий:

  • html страничку с версткой (layout.html)
  • папка css с единственным файлом стилей (style.css)
  • папка img с необходимыми графическими материалами использованными в верстке

Сроки выполнения

Один рабочий день
